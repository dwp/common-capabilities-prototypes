Create task template

Task templates are sets of data that describe common tasks.

For example, a task template might be created for the common “Find an address” task, that looks like this:

Task template name: Find an address
Priority: Medium
Task start page: https://gov.uk/find-address
Time for task completion: 1 hour

They can have characteristics associated that help match tasks to agents.

For example "Verify address for NI" or "Find an address" or "Reopened Claim".
