Glossary

List of terms used in this service

Characteristics

Characteristics are features that an agent or task may have (such as language, role, skill or location). Agents and tasks can be matched if they share the same characteristics.

Activities [or Task templates]

Activities [or Task templates] are repetitive tasks that can be created simply by submitting the appropriate subject key or citizen GUID (global unique identifier).

Agent profiles

Agent profiles are profiles of staff with a particular set of skills who are able to work on tasks. They can belong to one or more business areas and be part of an agent group.

Agent groups

Agent groups are made up of staff who have the same skills in their agent profile. Agents can claim tasks as members of a group using their group attributes.
